until Loop:

It will execute /run until the condtion becomes true.

1>5 -> false //
2>5 ->false // it will execute 
3>5
4>5
5>=5
6>5// True -> it will come out

------------------------------------------------------------

syntax:

until [ condtion ]
do
  //block of code
done

---------------------------------------------------------------------------------------------------

#!/bin/bash
n=1

until [ $n -gt 10 ]
do
        echo "$n"
        n=$(( $n+1 ))
done
